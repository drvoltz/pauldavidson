# Paul Davidson – Faculty Website (Static HTML)

This is a clean, low-maintenance academic website you can deploy via GitHub Pages.

## Deploy (GitHub Pages)
1. Create a repo (e.g., `pauldavidson.github.io`).
2. Upload the contents of the `paul-faculty-site/` folder to the repo root.
3. GitHub → Settings → Pages → Deploy from `main` branch.

## Customize
- Replace placeholder links marked “add link” (Scholar/ORCID/LinkedIn).
- Update `contact.html` with your real email + office.
- Replace the placeholder CV at `assets/PaulDavidson_CV.pdf`.
- Update Home page highlights and selected papers (1–2×/year is enough).

## Files
- `index.html` – Home
- `research.html` – Research themes
- `vision.html` – 10-year vision
- `publications.html` – curated papers + Scholar link
- `students.html`, `talks.html`, `service.html`, `teaching.html`
- `cv.html`, `contact.html`
- `assets/style.css` – global style
